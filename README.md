Project 2: RPS Arena – Rock Paper Scissors Game
Description
A console-based game where the player competes against the computer by selecting Rock, Paper, or Scissors. The computer generates a random choice and the winner is determined according to the game rules.
Tools / Resources
• Language: Java (JDK 8+)
• Libraries: java.util.Random, java.util.Scanner
• IDE: IntelliJ IDEA / Eclipse
Steps
Display the available choices to the player.
Accept the player's selection.
Generate a random choice for the computer.
Compare both choices using game rules.
Display the winner and allow replay.
Architecture
Decision-Based Architecture:
Player Choice → Computer Choice Generator → Rule Evaluation Engine → Result Display
Outcome
An entertaining game that introduces decision-making logic, randomization, and user interaction.
